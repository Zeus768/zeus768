CATEGORY IMAGES FOLDER - Strike Zone Addon
==========================================

Place your custom category PNG images here with these exact filenames:

PLACEHOLDER NAMES (use these exact names):
------------------------------------------
ufc.png              -> UFC category
mma.png              -> MMA category  
boxing.png           -> Boxing category
kickboxing.png       -> Kickboxing category
k1.png               -> K-1 category
bellator.png         -> Bellator category
one_championship.png -> ONE Championship category
pfl.png              -> PFL category
bkfc.png             -> BKFC category
cage_warriors.png    -> Cage Warriors category

Supported formats: PNG (recommended), JPG, JPEG

IMAGE RECOMMENDATIONS:
- Size: 512x512 pixels (square) or 1920x1080 for fanart-style
- Format: PNG for best quality with transparency support
- Style: Match your addon theme/fanart style

If no custom image is found for a category, the addon will use fanart.jpg as default.
