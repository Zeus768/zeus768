# Zeus768 Repository

Official Kodi repository for **Orion Media Explorer** addon.

![Zeus768 Repository](https://raw.githubusercontent.com/zeus768/zeus768/main/repository.zeus768/fanart.jpg)

## Installation

### Method 1: Install Repository (Recommended)
1. Download the repository zip: [repository.zeus768-1.0.0.zip](https://github.com/zeus768/zeus768/raw/main/repository.zeus768/repository.zeus768-1.0.0.zip)
2. In Kodi, go to **Add-ons** > **Install from zip file**
3. Navigate to the downloaded zip and select it
4. After installation, go to **Install from repository** > **Zeus768 Repository**
5. Select **Video add-ons** > **Orion** > **Install**

### Method 2: Direct Install
1. Download the addon zip: [plugin.video.orion-2.3.0.zip](https://github.com/zeus768/zeus768/raw/main/plugin.video.orion/plugin.video.orion-2.3.0.zip)
2. In Kodi, go to **Add-ons** > **Install from zip file**
3. Navigate to the downloaded zip and select it

## Features

### Browse by Streaming Service
- Netflix, Disney+, Amazon Prime Video, Max (HBO)
- Hulu, Paramount+, Apple TV+, Peacock
- Crunchyroll, Tubi TV, Pluto TV, and 20+ more
- Provider icons and region selection

### Multi-Source Scraping
- **Stremio Addons**: Torrentio, MediaFusion
- **Torrent Indexers**: Orionoid, YTS, EZTV, 1337x, TorrentGalaxy
- **Direct Streams**: VidSrc, FlixMomo, HydraHD, Cineby, and more
- **DDL**: RLSBB (Rapidgator, NitroFlare)

### Debrid Support
- Real-Debrid
- Premiumize
- AllDebrid

### Additional Features
- Trakt integration (sync, watchlist, history)
- TMDB metadata (posters, plots, ratings)
- Quality filtering (4K, 1080p, 720p, SD)
- Color-coded source types
